Solon 

Solon is solitair game with board edit.
Solitaire - jump over pegs to clear board
OnOff - switch white and black pegs
Shifter - shift pegs as you want


Menu:

F File	usual file handling ...
E Edit	
	Invert	Change black and white pegs.

Mouse:
Left button draws, Right button erases.
Middle button makes zoom.

Edit modes:

  Play		Play the board
  Free		Click change peg, move draw with peg on mouse down
  Line		Draw line
  Rectangle	Filled rectangle
  Circle rp		Circle given by center and point on circle
  Circle 2p	Circle given by 2 points on circle
  Fill		Fill to boundaries
  Block		Add block between 2 cells
  Back		Set background color (colors are not saved)
  Fore		Set foreground color

Mouse:

Left	Play
Wheel	Zoom,Ctrl or Shift Move,Ctrl and Shift rotate

Keys:
Ctrl+Z	undo
Ctrl+Y	redo

Export
                       
bitmap : png 
vector : svg,pdf (uncompressed)


It is free for any use.
